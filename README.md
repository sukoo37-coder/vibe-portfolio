# Vibe Portfolio

## 현대적인 포트폴리오 웹사이트

반응형 디자인의 현대적 포트폴리오 웹사이트입니다.

### 기능
- 반응형 디자인 (모바일, 태블릿, 데스크톱)
- 부드러운 애니메이션과 트랜지션
- 인터랙티브 네비게이션
- 스킬 바 애니메이션
- 타이핑 효과
- 문의 폼

### 기술 스택
- HTML5
- TailwindCSS
- Vanilla JavaScript
- Font Awesome

### 사용 방법
1. 저장소 클론
2. `index.html` 파일을 브라우저에서 열기

### 라이브 데모
곧 GitHub Pages로 배포 예정
